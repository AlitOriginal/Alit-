🚀 БЫСТРЫЙ СТАРТ (3 ШАГА)
══════════════════════════════════════════════════════════════════════════════

Шаг 1: Установите зависимости
────────────────────────────────────────────────────────────────────────────
Windows:
  Откройте консоль и выполните:
  pip install -r requirements.txt

Шаг 2: Добавьте API ключ OpenAI
────────────────────────────────────────────────────────────────────────────
1. Откройте файл: script.js
2. Найдите строку: this.apiKey = 'sk-proj-...'
3. Замените на ваш ключ с https://platform.openai.com/api-keys

Шаг 3: Запустите приложение
────────────────────────────────────────────────────────────────────────────
Windows:
  python manage_server.py
  (выберите "1. Запустить сервер")

macOS/Linux:
  python3 manage_server.py
  (выберите "1")

Браузер откроется автоматически на http://localhost:5000
Или откройте вручную: index.html

🐛 ЧАСТО ВСТРЕЧАЕМЫЕ ВОПРОСЫ
══════════════════════════════════════════════════════════════════════════════

Q: Как добавить API ключ?
A: Откройте script.js и замените 'sk-proj-...' на ваш ключ

Q: Port 5000 уже занят?
A: Измените port в server.py (строка с app.run)

Q: Python не найден?
A: Добавьте Python в PATH при установке

Q: Сервер не запускается?
A: Проверьте антивирус, перезагрузитесь

Q: Не видны сообщения?
A: Перезагрузите страницу браузера

Q: Как сбросить пароль?
A: Удалите пользователя в manage_server.py
