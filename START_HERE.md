# 🎉 AI Chat Assistant - ГОТОВО К ЗАПУСКУ!

Полнофункциональное веб-приложение с AI чатом, регистрацией пользователей и мультиплеер функциями.

## ⚡ БЫСТРО НАЧНИТЕ (2 МИНУТЫ)

### 1️⃣ Установите зависимости
```bash
pip install -r requirements.txt
```

### 2️⃣ Добавьте API ключ OpenAI
Отредактируйте `script.js`:
```javascript
this.apiKey = 'sk-proj-ВАШ_КЛЮЧ_ЗДЕСЬ';
```

### 3️⃣ Запустите приложение
```bash
python manage_server.py
# Выберите "1. Запустить сервер"
```

### 4️⃣ Откройте браузер
```
http://localhost:5000
```

**Готово! 🎉 Начните использовать приложение!**
